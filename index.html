About me

I am a junior developer who enjoys working in front-end development but also gets a kick out of back-end development.

After working in various customer service roles for the past 10 years, I was seeking a career change where I would be continuously challenged. The idea of Software Development originated from my intention of creating an online drop-shipping store, when I realised I enjoyed customising my Shopify website theme more than anything else.

After going a self-taught route for a while, I enrolled in a self-funded technical training course which later led me to a twelve-week intensive bootcamp, where I completed projects using HTML, CSS, JavaScript, MySQL, MongoDB and Python.

When I'm not coding I'm usualy exploring the city, learning something new or running (after a toddler).

Breathe and Relax App

A guided breathing/ambient sounds app. The user can pick the amount of time they want to use it and choose different ambient sounds. The app will let the user know when to breath in/hold/breathe out. This project was created using HTML, CSS and JavaScript.

Vegan Wine Checker

CRUD app where a user can search wines and see if they're vegan friendly. It has a database of almost 6000 wines, and can be updated by the user (add, delete, update functions as well as search). This project was created using Python, Pandas, MySQL, Flask and Bootstrap.
